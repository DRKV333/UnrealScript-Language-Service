This work-in-progress extension for Visual Studio Code, adds language support for UnrealScript, including:

    Auto-Completion
    Symbol Searching
    Symbol Renaming
    Find All References
    Go to Definition, Peek Definition
    Quick Info (Hover), Error Squiggles
